Разработать веб-приложение на PHP с использованием базы данных MySQL для хранения и управления коллекцией книг. Приложение должно обеспечивать добавление, редактирование, удаление и просмотр информации о книгах.


Функциональные требования

Создайте таблицу для хранения информации о книгах с полями:

id (INT, PRIMARY KEY, AUTO_INCREMENT) — уникальный идентификатор книги.
title (VARCHAR(255)) — название книги.
author (VARCHAR(255)) — автор книги.
year (YEAR) — год издания.
genre (VARCHAR(100)) — жанр книги.
description (TEXT) — описание книги.

Функционал CRUD
Реализуйте следующие функции:

Создание: Форма для добавления новой книги в базу данных.
Чтение: Страница со списком всех книг 

Обновление: Возможность редактировать информацию о книге.
Удаление: Возможность удалять книги из базы.
Интерфейс приложения должен быть минималистичным и удобным для пользователя.
При добавлении или редактировании книги поля должны проверяться на заполненность.(Javascript)
